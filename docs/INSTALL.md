# Installation

This fork publishes Windows builds only.

## Windows

1. Open [Releases](https://github.com/Leostrange/dlman/releases).
2. Download the latest `.exe` or `.msi` installer.
3. Run the installer.
4. Select `Русский` in the app settings if needed.

## Unsupported Targets

This fork does not publish installers for macOS or Linux.
